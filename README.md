# flauschige-uhr

flauschige-uhr is a clone of the kde fuzzy-clock widget.

## Arguments:

`-j` how jessica likes her time

`-g` how guddaff likes his time

`-w` how wucke likes his time

## Install

`make` to build the project

`make install` to install it
